Computational Metamaterials

Computational metamaterials are engineered materials that perform computations, like integration or differentiation, on waveforms propagating through them. Analog computing devices have been implemented previously, but they are usually too large to be practical. However, by utilizing metamaterial concepts, material properties can be precisely controlled, enabling the miniaturization of analog computing components. These notes on computational metamaterials come primarily from [1]

Metasurface Approach

This approach to realizing a computational metamaterial requires material blocks that apply a fourier transform to the incoming waveform. It is possible to construct this in the forward direction, from spacial domain to frequency domain. However, the reverse direction requires negative permittivity/permeability materials which are much less practical.

Greens Functions, Math Operator and Spacial Fourier Transform

To apply the mathematical operator, properties of Greens function convolution and the spacial Fourier transform are used. If you take the convolution between the Green's function associated with the math operator and the input function, the resulting function should be the input function with the operator applied.

\cd{g}(y)=\int_{-\infty}^\infty \cc{f}(u)\ca{G}(y-u)\,du
\cc{f}(y): \text{Input Function} \phantom{\tilde{G}(k_{y,i})} \cd{g}(y): \text{Output Function} \phantom{\tilde{G}(k_{y,i})} \ca{G}(y): \text{Greens function} \phantom{\tilde{G}(k_{y,i})}

Using a convolution in this way is beneficial because when this problem is taken into k space, it becomes multiplication instead of integration.

\cd{\tilde{g}}(k_y)= \ca{\tilde{G}}(k_y)\cc{\tilde{f}}(k_y)
\tilde{A}: \text{Spacial Fourier transform of } A \phantom{\tilde{G}(k_{y,i})} \cc{\tilde{f}}(y): \text{Input Function} \phantom{\tilde{G}(k_{y,i})} \cd{\tilde{g}}(y): \text{Output Function} \phantom{\tilde{G}(k_{y,i})} \ca{\tilde{G}}(y): \text{Greens function} \phantom{\tilde{G}(k_{y,i})}

Below is the equation to take the spacial Fourier transform from y to k_y

\cc{\tilde{f}}(k_y)=\int_{-\infty}^\infty \cc{f}(y)e^{-jk_y y}\,dy
